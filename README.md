## 📌 Overview

This project focuses on predicting passenger survival on the Titanic using Machine Learning techniques. The dataset contains information about passengers such as age, gender, class, fare, and more.

The project follows a complete workflow including **data preprocessing, exploratory data analysis (EDA), feature engineering, model building, evaluation, and prediction**.

---

## 🎯 Project Objective

* Analyze Titanic dataset to understand survival patterns
* Identify important features affecting survival
* Build classification models to predict survival
* Improve data analysis and machine learning skills

## 📂 File Included

* `Titanic project.ipynb` – Jupyter Notebook with full analysis and model building
